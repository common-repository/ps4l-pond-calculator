WDS Pond Calculator to calculate an estimated volume of a pond to help find a pond liner that fits.

== Description ==

WDS WordPress Pond Calculator will help you find the best fitting pond liner for your pond.  With this simple yet effective plugin you can calculate the volume of several types of ponds using the various tabs for Square, Over or Round.

### Features

* Change the colors to match your website perfectly
* Free install onto your website
* Built in short codes to allow you to add it to your posts or pages
* Widget for sidebar use
* Built to be responsive to several screen sizes

== Installation ==

1. Upload `wds-wordpress-pond-calculator` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
